Installation

Head to http://friiishop.us.to/ and click the application download link for your os wait for the application to download then unzip the application.

Using the application

Once you have installed the application for your os move the application to a memorial place preferably your desktop. Then click or double click the app to start it. Once it has boded select your consoles region then select the console you want to see all games for it will take about 30 seconds to load then follow on screen instructions
